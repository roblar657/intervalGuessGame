# Interval gjette spill app

En gjetter ett tall i mobil appen, og sender tallet til web-server (se vedlagt .jsp fil). I mobil app, så får en hjelp og støtte
til å huske hvilke interval som inderholde det riktige tallet. En kan gjette opp til det antall som web-server aksepterer (forhåndinnstilt til 50).
